Step 5: Effects

JSTween effects apply to motion tweens such as left and right. They allow your animation to seem smoother or rougher depending on the effect selected. A complete list of motion effects is available at the JSTween website. For our purposes, we will focus on the two most noticeable motion effects: bounceInOut and elasticInOut.

BounceInOut:

Test Code
  • $('#luke').tween({
    • left: {
      • start: 0,
      • stop: 175,
      • time: 0,
      • units: 'px',
      • duration: 1,
      • effect: 'bounceInOut'
    • }
  • });
  • $.play();

bounceInOut means that you are adding the "bounce" effect to the beginning and the end of the motion tween. There are separate bounceIn and bounceOut effects if you only want the effect applied to the beginning or the end of a motion tween.

ElasticInOut:

Test Code
  • $('#luke').tween({
    • left: {
      • start: 0,
      • stop: 175,
      • time: 0,
      • units: 'px',
      • duration: 1,
      • effect: 'elasticInOut'
    • }
  • });
  • $.play();

elasticInOut means that you are adding the "elastic" effect to the beginning and the end of the motion tween. There are separate elasticIn and elasticOut effects if you only want the effect applied to the beginning or the end of a motion tween.

If you have a good grasp of effects, move on to Step 6.
