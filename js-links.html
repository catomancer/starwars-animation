Step 2: Set Up Libraries

The JSTween library requires jQuery, so you will need to either download jQuery or link to it. You can download your favorite version from the jQuery website, or you can link to Google's hosted jQuery libraries by following the instructions at developers.google.com.

Important Note: If you choose to link to a Google library, note that Google will give you a link in the following format: //ajax.googleapis.com/ajax/.../jquery.min.js. REMEMBER to add http: or your program won't link to the jQuery library and nothing will happen.

To animate your DIVs, you will need to download the JSTween library. You can either visit the JSTween website and click "Download" on the right-hand side, or you can use this direct link. Unzip the JSTween files into your chosen directory. You can use either library: jstween-x.js is the uncompressed, easy-to-read version for development, and jstween-x.min.js is the compressed, impossible-to-read version for production. Basically, if you're going to be building a library based on JSTween, use jstween-x.js, but for our purposes, jstween-x.min.js will work just fine.

Important Note: When you link to your libraries, jQuery MUST be the first script loaded, otherwise JSTween will try to interpret your jQuery as pure JavaScript. It's not pretty and nothing will work on your page. You will be sad. Don't do it.

How to link to jQuery and JSTween:

<script src="http://ajax.googleapis.com/ajax/libs/jquery/2.1.1/jquery.min.js"></script> /* FIRST */
<script src="jstween-1.1.min.js"></script>

You can test that jQuery is working properly by adding the following to your HTML file:

  • <script type="text/javascript">
    • $(document).ready(function(){
      • $("div").css("border", "3px solid red");
    • });
  • </script>

If jQuery is not working, follow the steps in this helpful guide from CSS-Plus.com. The above test was taken from that website.

Once you have verified that jQuery is working, delete the test script and move on to Step 3.
