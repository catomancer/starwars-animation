Now you're all ready to battle! I chose to implement my animation using JavaScript functions, both of which are listed in the scrolling code box below. I created two DIVs to act as "start" and "reset" buttons, and the corresponding function starts execution when the button is clicked. For example, my "start" button HTML looks like this:

<div class="button" id="start" onclick="startBattle">Start Battle!</div>
