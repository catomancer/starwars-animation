Step 1: Format

The most important part of creating the Force Battle animation is getting the elements set up correctly. Luke and Vader are each individual DIVs with a background image. The Emperor's command room is set as the background image of the animation container DIV.

HTML:

  • <div id="finished">
    • <div id="luke"></div>
    • <div id="vader"></div>
  • </div>

CSS:

  • #finished {
    • width:640px;
    • height:480px;
    • margin:5px auto auto auto;
    • background-color:#000000;
    • background-image:url('background.png');
    }
  • #luke {
    • width:151px;
    • height:194px;
    • margin:175px auto auto 5px;
    • float:left;
    • background-color:transparent;
    • background-image:url('luke.png');
    • display:inline-block;
    • position:relative; /* IMPORTANT!! Browser will hold DIV in place otherwise */
    }
  • #vader {
    • width:222px;
    • height:239px;
    • margin:150px 5px auto auto;
    • float:right;
    • background-color:transparent;
    • background-image:url('darthvader.png');
    • display:inline-block;
    • position:relative; /* IMPORTANT!! Browser will hold DIV in place otherwise */
    }

The HTML is very simple: a container DIV called "finished", and character DIVs for Luke and Vader. The CSS determines how well they line up with each other and the chosen background image.

It is better to use PNG images for your DIVs because they can have a transparent background. JPEG images interpret a transparent background as white, which would leave a big white box around each of your characters.

For the Luke and Vader DIVs, REMEMBER to set their positions to "relative". If you don't, none of the JSTween animations will work because the browser will hold the DIVs in place. You will tear your hair out trying to figure out the problem and you will be sad. Don't do it.

At this point, you should have a nice 640x480 box on your screen with your chosen background image, and Luke and Vader should be aligned so that they are facing each other on opposite sides of the box.

If everything is displaying properly, you should move on to Step 2.
