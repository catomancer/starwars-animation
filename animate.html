Step 3: Animate

For each of the examples below, copy and paste the code into your webpage to test it. You can embed it with the <script> tag, or you can link to a separate JavaScript file. You can choose to trigger the animation when a button is clicked, or you can have it load as soon as the document is ready using jQuery. I used the first method, which I will detail in the last step.

Important: What makes the magic happen here is the $.play() command at the end of each example. Without it, JSTween won't know to display the animation and nothing will happen.

For each of these examples, we will focus on the start and stop attributes. With start, you set where you want your animation to start. This can be a position on the screen, an opacity, a rotation, or pretty much anything depending on the type of animation called. stop functions the same way, except it's where you want your animation to stop. It's important to specify units so that JSTween knows what you're talking about.

For now, don't worry about time, duration, and effect, as they are covered in the next steps.

Move:

Test Code
  • $('#luke').tween({
    • left: {
      • start: 0,
      • stop: 175,
      • time: 0,
      • units: 'px',
      • duration: 1,
      • effect: 'easeInOut'
    • }
  • });
  • $.play();

Here, I am telling JSTween to move Luke from the left, and to start him at 0 and stop him at 175. I specify units: 'px' so that JSTween knows to move Luke 175 pixels, as opposed to percent or degrees.

Glow:

Test Code
  • $('#luke').tween({
    • shadow: {
      • start: '0px 0px 0px #000',
      • stop: '10px 10px 30px #00f',
      • time: 0,
      • duration: 0.5,
      • effect: 'easeInOut'
    • }
  • });
  • $.play();

shadow is short for "box-shadow" and functions the same way as the CSS3 box-shadow effect. The first two values are the horizontal and vertical position of the shadow, the third is the blur distance, and the #000 sets the color of the shadow. Since the units are already specified, you don't need an explicit units attribute. For a more detailed explanation of box-shadow syntax, go to w3schools.com.

Rotate:

Test Code
  • $('#luke').tween({
    • rotate: {
      • start: 0,
      • stop: 360,
      • units: 'degrees',
      • time: 0,
      • duration: 1,
      • effect: 'easeInOut'
    • }
  • });
  • $.play();

rotate is fairly self-explanatory: set your start and stop values and remember to specify your units as 'degrees'. To rotate the opposite direction, input a negative stop value.

Fade:

Test Code
  • $('#luke').tween({
    • opacity: {
      • start: 100,
      • stop: 0,
      • time: 0,
      • duration: 1,
      • effect: 'easeInOut'
    • }
  • });
  • $.play();

In this example, units are not technically necessary as JSTween automatically interprets them as percents, though you can include them for clarity.

Test these examples on your computer and make sure you understand how they work. Then you can move on to Step 4.
